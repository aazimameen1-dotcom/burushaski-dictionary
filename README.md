# Burushaski-Kashmiri Dictionary

A comprehensive digital dictionary bridging the Burushaski and Kashmiri languages, featuring dialectal variations from Srinagar and Gilgit-Baltistan regions.

## About

The Burushaski-Kashmiri Dictionary is a digital resource documenting the Burushaski language, an ancient and endangered language isolate spoken in the mountainous regions of northern Pakistan. This project provides bilingual entries with phonetic transcriptions, grammatical information, and contextual examples.

### Features

- **Two Dialect Variants**: Srinagar dialect (Hari Parbat) and Gilgit-Baltistan dialect (Hunza/Nagar)
- **Comprehensive Entries**: Each entry includes Burushaski word, Persian script, IPA pronunciation, English definition, Kashmiri equivalent, and usage examples
- **Interactive Search**: Filter by part of speech (nouns, verbs, adjectives, etc.)
- **Dialect Switching**: Compare entries between different Burushaski dialects
- **Audio Support**: Text-to-speech functionality for pronunciation
- **Citation Tools**: Copy formatted citations for academic use

## Usage

1. Open `index.html` in a modern web browser
2. Use the search bar to find words in Burushaski, English, or Kashmiri
3. Switch between dialects using the dropdown in the header
4. Filter entries by part of speech using the filter buttons
5. Click the speaker icon to hear pronunciation (requires browser TTS support)

## Dialect Information

### Srinagar Dialect (Hari Parbat)
- Focuses on the variant spoken in Srinagar's Hari Parbat area
- Exhibits vowel centralization (e.g., /a/ → /ʌ/) compared to Hunza forms
- Primary informants: Mr. Raja Zeeshan, Mr. Raja Yawar
- Academic framework: Dr. Sadaf Munshi (University of North Texas)

### Gilgit-Baltistan Dialect (Hunza/Nagar)
- Represents the dialect spoken in Hunza and Nagar valleys
- Based on comprehensive lexicographic works by Allamah Nasiruddin Nasir Hunzai
- Sources: *Burushaski-Urdu Lughat* (Vols 1, 2, 3) and *Buruśo-Birkiş*
